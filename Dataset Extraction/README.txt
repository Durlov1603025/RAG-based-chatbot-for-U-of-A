How to Run the Program
======================

Follow these steps to run the web scraping script (scrap.py):

1. Make sure Python 3.13.1 or greater is installed.

2. Open the Backend folder from VSCode.

3. Create a Virtual Environment
-------------------------------
Open your terminal and run:

    python -m venv venv

This will create a virtual environment named 'venv' in your current directory.

4. Activate the Virtual Environment
-----------------------------------
To activate the virtual environment write the following command in the terminal:

On Windows:
    venv\scripts\activate

On macOS/Linux:
    source venv/bin/activate


5. Install Required Packages
----------------------------
With the virtual environment activated, install the required dependencies by writing the following command in the terminal:

    pip install -r requirements.txt


6. Run the Script
-----------------
Now run your Python script by writing the following command in the terminal:

    python scrap.py


Enjoy scraping!
